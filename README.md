# pycoplayer
Pycoplayer is a general AI that learns how to play games through trial and error, based on the visual and auditory output from any game of your choosing, it delivers keyboard, mouse, and controller input to play.

## Visualization

A simple script is included to display the dependencies between modules in this project. Install the optional dependencies and run `python -m src.visualize` to open a window showing the module graph.

```bash
pip install networkx matplotlib
python -m src.visualize
```
